En este OVI se pretende que los estudiantes adquieran los conocimientos necesarios para el desarrollo de software por medio de los recursos de información brindados, tales como: teóricos, páginas web, blogs, videos, etc; con los cuales, los estudiantes puedan aplicar esta información para el desarrollo de software.

Objetivo General

Conocer e implementar un motor de base de datos, para almacenar la información de forma correcta y eficiente.

Objetivos Especificos

  • Identificar los conceptos básicos de las bases de datos relacionales.
  • Instalar y administrar de forma eficiente una base de datos relacional.
  • Diferenciar cada una de las plataformas que ofrece el mercado para las bases de datos.
