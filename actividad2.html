Creación de Tablas en SQL Server

Una tabla es una estructura de datos que organiza los datos en columnas y filas; cada columna es un campo (o atributo) y cada fila, un registro. La intersección de una columna con una fila, contiene un dato específico, un solo valor.

Cada registro contiene un dato por cada columna de la tabla.
Cada campo (columna) debe tener un nombre. El nombre del campo hace referencia a la información que almacenará.
Cada campo (columna) también debe definir el tipo de dato que almacenará.

Primer problema:

Necesita almacenar los datos de sus amigos en una tabla. Los datos que guardará serán: apellido, nombre, domicilio y teléfono.

Elimine la tabla "agenda" si existe:

if object_id('agenda') is not null drop table agenda;

Intente crear una tabla llamada "/agenda":

create table /agenda( apellido varchar(30), nombre varchar(20), domicilio varchar(30), telefono varchar(11));

Aparece un mensaje de error porque usamos un caracter inválido ("/") para el nombre.

Cree una tabla llamada "agenda", debe tener los siguientes campos: apellido, varchar(30); nombre, varchar(20); domicilio, varchar (30) y telefono, varchar(11):

CREATE TABLE agenda( apellido varchar(30), nombre varchar(20), domicilio varchar(30), telefono varchar(11) );
